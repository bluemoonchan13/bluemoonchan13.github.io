Muffin Top

I created this project to be mostly for fun and I was also learning about certain programs to create this. I created a top that is can be 3D printed and be a functioning top. The top can be spun for a good minute stopping. The complications were the shapes used in the project. I did not use a simple shape of a circle but used an oval to create the shapes which put the ridges of the muffin liner at odd angles. The overall project came together once the liner was in place.

The project was made for just enjoyment and for learning purposes on my part and I ended up with something that I still spin today.
